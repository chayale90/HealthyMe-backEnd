Clothes API Documentation

Clothes Routes

Method URL Body Description
GET /foods Get a list of all the foods, will present 6 results per page
GET /foods?page=2 Get a list of all the foods, will present 6 results in page 2
GET /foods?sort=totalPrepMinutes Get a list of all the foods sort by totalPrepMinutes, will present 6 results per page
GET /foods?reverse=yes Get a list of all foods sorted in descending order, will display 10 results per page
